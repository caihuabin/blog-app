## 运行环境

* PHP 5.5.9+
* Laravel 5.1+

## 安装使用

需使用apache或nginx部署服务器.

### 1. Clone 项目

    git clone https://github.com/icyse/carte

### 2. Composer install

    cd blog-app
    composer install
    
### 3. Database建立

配置数据库信息, 然后: 

    php artisan migrate
